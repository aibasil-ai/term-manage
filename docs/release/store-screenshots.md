# Chrome Web Store 截圖清單

以下檔案已生成於 `dist/store-screenshots/`，尺寸皆為 `1280x800`：

1. `01-overview.png`
   - 主畫面總覽：顯示已建立項目、分類標籤與快捷鍵綁定。
2. `02-create-item-form.png`
   - 新增項目流程：展示標題、分類、快捷鍵槽位與內容編輯。
3. `03-search-and-filter.png`
   - 搜尋與分類篩選：展示快速查找指定內容項目。
4. `04-edit-existing-item.png`
   - 編輯既有項目：展示既有資料回填與更新流程。
5. `05-display-mode-and-transfer.png`
   - 顯示模式與資料傳輸：展示模式切換與匯出/匯入區塊。

## 重新產生

```bash
node scripts/generate-store-screenshots.mjs
```

輸出路徑：`dist/store-screenshots`
