文字快填助手

這是一個專為「重複輸入文字內容」場景打造的 Chrome Extension。你可以先在擴充功能內建立常用片段（例如客服回覆、測試帳號資訊、FAQ 段落、固定格式內容），再一鍵帶入任意網站輸入框，降低重工、提升一致性與輸入速度。若需檢視隱私權內容，可前往 隱私權政策頁面。

核心用途

專案核心目標是把「手動重複打字」轉成「管理好內容後快速貼入」。透過集中管理文字片段，團隊或個人可在不同網站快速套用一致用語，減少錯字與遺漏，尤其適合需要大量回覆或固定格式輸入的工作流程。

立即安裝擴充功能 前往 Chrome Web Store 安裝「文字快填助手」。 點擊後會開啟新分頁

主要功能

  • 建立、編輯、刪除內容項目（標題 / 分類 / 內容）
  • 關鍵字搜尋與分類篩選
  • JSON 匯出與匯入（可備份與還原）
  • 點擊「帶入」將內容插入當前輸入欄位
  • 支援快捷鍵插入最近使用項目或指定槽位項目

顯示模式

為了貼合不同使用習慣，介面支援三種開啟方式：附著在 Chrome 側邊欄、工具列彈出小窗、獨立視窗。你可以依當下工作流切換，維持輸入與查找內容的效率。

操作流程

  1. 在擴充功能建立內容項目與分類。
  2. 切到目標網站並先選取輸入框。
  3. 回到擴充功能列表點擊「帶入」。
  4. 或直接使用已綁定的快捷鍵插入。
  5. 必要時用匯出功能做備份。

資料儲存與同步

項目資料預設儲存在 chrome.storage.sync，可在同一 Google 帳號的 Chrome 裝置間同步；若環境不支援，會回退至 chrome.storage.local。專案本身不建立開發者後端資料庫。

帶入限制

  • 僅在你主動操作時執行帶入
  • 無法在 chrome://* 等受保護頁面注入
  • 需先聚焦目標輸入欄位再插入

快速開始

  • npm install
  • npm test
  • 到 chrome://extensions 載入未封裝項目

技術基礎

  • Chrome Extension Manifest V3
  • 原生 JavaScript 模組化結構
  • 可測試的插入核心邏輯（Vitest）
