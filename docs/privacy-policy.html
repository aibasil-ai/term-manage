隱私權政策

最後更新日期：2026-02-24

本隱私權政策適用於「文字快填助手」Chrome 擴充功能（以下稱「本擴充功能」）。

1. 我們蒐集哪些資料

本擴充功能不會主動蒐集或傳送可識別個人身分的資料到開發者伺服器。

本擴充功能僅在你的瀏覽器本機或 Chrome 同步儲存空間中保存下列資料：

  • 你自行建立的文字項目（標題、分類、內容）
  • 使用偏好（例如顯示模式）
  • 快捷鍵槽位綁定資訊
  • 最近一次使用項目 ID

2. 資料用途

上述資料僅用於提供本擴充功能核心功能：

  • 顯示與管理你的文字項目
  • 將你選擇的內容帶入目前網頁輸入框
  • 提供快捷鍵快速插入與跨裝置同步（若你啟用 Chrome 同步）

3. 資料儲存與傳輸

  • 資料儲存於 chrome.storage.sync（若可用）或 chrome.storage.local
  • 本擴充功能不建立開發者後端資料庫，不會主動將內容上傳至開發者伺服器
  • 若你使用 Chrome 同步，資料會依 Google Chrome 同步機制在你的裝置間同步

4. 權限說明

  • storage：儲存你的文字項目與設定
  • activeTab、scripting：在你主動操作時，將文字插入目前分頁輸入框
  • 網站存取權限（http://*/*、https://*/*）：讓擴充功能可在一般網站分頁執行帶入流程
  • sidePanel：提供側邊欄介面

5. 第三方分享

本擴充功能不出售、不出租、不交易你的資料給第三方。

6. 使用者控制權

你可透過以下方式管理資料：

  • 在擴充功能內編輯、刪除項目
  • 使用匯出/匯入功能自行備份與還原
  • 移除擴充功能以停止使用

7. 政策變更

若本政策有重大變更，將更新「最後更新日期」並於商店頁或專案文件說明。
